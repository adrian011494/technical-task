# Technical Task

## Project Structure

### Folders: 
- `client` Client side code with Vue.
- `server` Server side code with Express 

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm start
```

### Run the server

```sh
npm run start:server
```

### Run the client 

```sh
npm run start:client
```

### Test

```sh
npm test
```
